## show-subtitles

### 介绍

自动加载视频同名文件的外挂字幕文件。支持`ass`与`srt`文件.

例如将要播放的资源路径为`/home/video.flv`，加载该插件后将会寻找同名的字幕文件`/home/video.ass`与`/home/video.srt`

### 参数

### 参数列表

| 参数    | 说明      | 默认值                   |
|-------|---------|-----------------------|
| fonts | 字体文件路径  | fonts                 |
| alpha | 字幕文件透明度 | 1；允许取值0.0-1.0范围值表达透明度 |
